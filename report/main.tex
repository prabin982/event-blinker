\chapter*{Abstract}
\addcontentsline{toc}{chapter}{\numberline{}Abstract}

The rapid growth of urban social life has created a significant gap between event organizers and potential attendees. Existing platforms often lack real-time updates, geospatial precision, and integrated transport solutions. This project presents \textbf{Event Blinker}, a comprehensive real-time event discovery platform that integrates three key innovations: (1) geospatial event discovery via interactive Mapbox maps, (2) an AI-powered virtual concierge (\textbf{Blinker AI}) leveraging LLAMA 3.3 via OpenRouter for localized event assistance, and (3) a peer-to-peer ride-sharing module enabling event-goers to book rides directly to venues.

The system is built on a robust technology stack comprising a Node.js/Express.js backend, PostgreSQL with PostGIS for geospatial queries, React Native (Expo) for the mobile application, Mapbox GL for map rendering, Socket.io for real-time communication, and React.js (Vite) for the organizer web portal. An additional admin portal handles event and rider verification. The ride-sharing module includes a Haversine-based distance calculator and NPR-based dynamic pricing.

Experimental results demonstrate sub-200ms API response times, sub-50ms WebSocket latency, and accurate geospatial filtering. The platform is deployed on Render and supports both Android and web clients.
