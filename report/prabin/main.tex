\chapter*{Abstract}
\addcontentsline{toc}{chapter}{\numberline{}Abstract}
This project presents the design and implementation of \textbf{Event Blinker}, a real-time discovery and mobility platform designed to unify the social event lifecycle with destination-integrated transport. The primary objective of the platform is to bridge the "informational and logistical gap" in modern urban environments, where citizens often face fragmented information about local events and inconsistent transport options for reaching those venues. By integrating geospatial discovery with a peer-to-peer (P2P) ride-sharing infrastructure, Event Blinker provides a seamless, end-to-end solution for urban navigation and social engagement.

The development process consists of several critical stages, including geospatial database modeling using PostGIS, real-time message orchestration via Socket.io, cross-platform mobile development using React Native, and the implementation of a decentralized bidding system for mobility. The system architecture is designed as a three-tier decoupled ecosystem, ensuring high scalability and reliability during surge loads. A lightweight AI concierge, powered by LLAMA 3.3 and grounded via a retrieval-augmented generation (RAG) pipeline, is integrated to provide context-aware support for venue-specific queries.

The platform is evaluated across multiple performance metrics, including spatial query latency, WebSocket broadcast delay, and system throughput. Experimental results demonstrate a high degree of efficiency, with mean spatial retrieval times recorded at 22ms and message broadcast latencies remaining under 40ms even under simulated load. The implementation of a multi-tier document verification pipeline (KYC) further ensures a safe and trusted environment for the peer-to-peer mobility exchange.

In conclusion, this work demonstrates that a unified approach to event discovery and mobility can significantly enhance urban accessibility. Event Blinker provides a technically rigorous foundation for future smart city informatics, offering a practical solution for making cities more navigable, social, and connected.
